# 🖥️ IT Asset Management System

## 📌 Description

This project is a full-stack IT Asset Management application.

* **Frontend**: React.js
* **Backend**: Spring Boot
* **Database**: MySQL

The application allows managing IT assets, users, tickets, and maintenance operations.

---

## 📁 Project Structure

```
it-asset-management/
├── backend/   # Spring Boot API
├── frontend/  # React Application
```

---

## 🚀 How to Run the Project

### 🔹 1. Run Backend

```bash
cd backend
mvn spring-boot:run
```

Backend runs on:

```
http://localhost:8085
```

---

### 🔹 2. Run Frontend

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```
http://localhost:3000
```

---

## 🔐 Authentication

* Login with email and password
* JWT token is used for authentication
* Token is stored in localStorage
